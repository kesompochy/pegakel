# Pegakel

Pegakel is a web browser-based tool for editing pixel art sprite sheet. It is designed for self-hosting.

## Usage

```sh
$ git clone https://github.com/kesompochy/pegakel.git
$ cd pegakel
$ bun run build
$ bun start
```

## Development

```sh
$ bun dev
```

## Testing

```sh
$ cd client
$ bun test-w
$ cd ../server
$ bun test --watch
```
